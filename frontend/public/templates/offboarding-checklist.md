# Employee Offboarding Security Checklist

**Company:** [YOUR COMPANY NAME]
**Employee leaving:** [EMPLOYEE NAME]
**Last working day:** [DATE]
**Offboarding managed by:** [YOUR NAME]

---

Complete every item on the employee's last day (or before). Do not delay — former employees with active access are a top security risk.

| # | Task | Done by | Date completed |
|---|------|---------|----------------|
| 1 | **Disable user account** — Deactivate their email and login accounts (same day). Do not delete yet — you may need access to their data. | | |
| 2 | **Remove from shared drives/folders** — Revoke access to Google Drive, SharePoint, Dropbox, or any shared storage. | | |
| 3 | **Transfer document ownership** — Reassign ownership of their files, shared docs, and folders to their manager or replacement. | | |
| 4 | **Remove from SaaS tools** — Remove or deactivate their account in every tool they used: [LIST YOUR TOOLS, e.g., Slack, Trello, HubSpot, Xero, Canva]. | | |
| 5 | **Revoke VPN / remote access** — Disable VPN credentials and any remote desktop access. | | |
| 6 | **Collect devices** — Retrieve laptop, phone, USB drives, access cards, keys, and any other company-owned equipment. | | |
| 7 | **Change shared passwords** — Update any passwords the employee knew that are shared (Wi-Fi, admin accounts, shared logins). | | |
| 8 | **Remove from email groups** — Remove from distribution lists, team inboxes, and mailing groups. | | |
| 9 | **Revoke third-party access** — Check for any OAuth/SSO connections they set up (e.g., "Sign in with Google" on external services). | | |
| 10 | **Forward email (if needed)** — Set up email forwarding to their manager for a defined period: [e.g., 30 days]. | | |

---

## Notes

- **Timeline:** Items 1, 5, and 6 must be done on the last working day. The rest within 24 hours.
- **Friendly departures too:** Follow this checklist even when someone leaves on good terms. It's not about trust — it's about hygiene.
- **Keep a record:** Save a copy of this completed checklist in the employee's HR file.
