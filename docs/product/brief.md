# SMB Security Quick Check — Product Brief

## Goal
Help SMB owners fix the biggest cybersecurity risks in 30 minutes — without login or payment.

## Primary user
Business owner (non-technical)

## Core value loop
Assess → Fix → Track progress → Feel safer

## MVP 0 Scope (current)
- Public explanation of risks & impact
- Interactive security checklist
- Progress indicator
- Simple guidance per item
- Save progress locally in browser

## Out of scope (for later)
- User accounts
- Teams & org hierarchy
- Reassessments
- Phishing campaigns
- GDPR exports

## Constraints
- Free infrastructure only
- Very simple UX
- Plain language (no security jargon)
- Privacy-friendly

## Success criteria (MVP 0)
- User can complete checklist in under 30 minutes
- Progress is clear
- User understands why each step matters
